# VS Code user settings

## Setup

### VS Code

Clone this repo directly into `~/.config/Code/User`, such that `~/.config/Code/User/settings.json` exists.

### Cursor:

Symlink VS Code's settings:

```bash
ln -s ~/.config/Code/User/settings.json ~/.config/Cursor/User/settings.json

```
